# MediLab Backend

This is the Node.js/Express/MongoDB backend for the MediLab Diagnostics platform.

## Features
- JWT Authentication
- Razorpay Payment Integration
- Appointment Management
- Lab Test & Health Package Data APIs

## Setup
1. Run `npm install`
2. Configure `.env` with your MongoDB URI and Razorpay keys
3. Run `node server.js` or `npm run dev` (if nodemon is configured)
